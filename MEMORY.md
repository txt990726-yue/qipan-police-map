# MEMORY.md

## 项目背景

用户根据微信聊天记录提出需求：为公安局大厅触摸屏制作一个可点击的辖区展示系统。

原始想法是用 PPT，但沟通后确认 PPT 不适合长期触摸屏交互展示，因此改为本地静态网页。

## 用户确认过的需求

- 使用触摸屏。
- 页面按液晶大屏展示，重点适配 16:9。
- 展示字段包括：姓名、电话、照片、职务、负责区域。
- 客户反馈希望做成翻页式大屏：第一页简介，第二页辖区图，第三页金字塔分工。
- 选择辖区后进入对应地图，再选择社区或区域。
- 点击社区后弹出社区主任信息。
- 设计要求较高，需要像正式大厅互动屏，而不是简单网页。
